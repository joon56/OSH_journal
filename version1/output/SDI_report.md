## 1. 연구 목적 및 SDI 개념

본 연구는 기업이 규제 대응을 위해 **서류상 안전(형식적 안전관리, Formal)**에는 투자하면서도 **실제 사고를 줄이는 안전(실질적 안전관리, Substantive)**은 방치하는 *안전 형식주의(Safety Decoupling)* 현상을 수치화한다.

**안전괴리지수(SDI, Safety Decoupling Index)** = Substantive score − Formal score

| SDI 값 | 해석 |
|---|---|
| SDI > 0 | 실질이 형식보다 강함 → **진짜 안전** |
| SDI ≈ 0 | 형식·실질 균형 |
| SDI < 0 | 형식만 있고 실질 부족 → **안전 형식주의(Safety Decoupling)** |

### 2-3. SDI 계산식

```
F-composite_k = 그룹 k 변수들의 행 평균 (각 변수 0~1 정규화)
S-composite_k = 그룹 k 변수들의 행 평균
F_score = mean(F-composite_1 ~ F-composite_7)   # 7그룹 평균
S_score = mean(S-composite_1 ~ S-composite_7)   # 7그룹 평균
SDI     = S_score - F_score
```

### 2-4. 음이항 GLM(GLM-NegBin) 회귀

업무상 사고 건수는 **과잉 0(excess zeros)**을 포함하는 카운트 자료이므로 일반 포아송 대신 음이항 GLM(GLM-NegBin)을 적용한다.

| 업종 | 사고 0건 비율 | 모형 선택 근거 |
|---|---|---|
| 건설업 | 64.8% | 구조적 0(노출 없는 현장) + 표본 0 혼재 |
| 제조·서비스업 | 80.4% | 소규모 사무직 등 구조적 무사고 집단 비율 높음 |

**음이항 GLM 모형 구조 (log link, offset=log(종사자수)):**

```
log(E[Y]) = Xβ + log(종사자수)   [log link, offset=log_workers]
Var(Y)   = μ + α·μ²               [음이항 분산: Poisson보다 overdispersion 허용]

```

- **설명변수**: SDI + 제도적 통제변수(formal_inst 등) + 근로자 구성 비율 + 범주형 통제변수
- **추정 방법**: BFGS 최적화, 최대 500회 반복
